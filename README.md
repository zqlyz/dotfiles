# Toffy's Dotfiles

## Usage

firstly ,install zsh and oh-my-zsh.

```bash
git clone --recursive https://github.com/zqlyz/dotfiles.git
cd dotfiles
./setup.sh
```

then use vim


```
:PlugInstall
```
