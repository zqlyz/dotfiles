# Toffy's Dotfiles

## Usage
```bash
git clone --recursive https://github.com/zqlyz/dotfiles.git
cd dotfiles
./setup.sh
```

then use vim


```
:PluginInstall
```

## Needed
These are some software need to be installed.
* i3
* feh
* rofi
* noto-sans-cjk
* font-awesome
* nerd-font(Source Code Pro)
* vim
* compton
* zsh (oh-my-zsh)
* polybar
* ranger
* w3m
